# 🗃️ The Prigozhin Archive

A bilingual digital corpus of interviews, documents, and statements related to Evgeny Prigozhin. This project aims to preserve and index publicly available content, allowing full-text search in both Russian and English, date-based filtering, and access to scanned or uploaded media (PDFs, images).

---

## 🚀 Features

- ✅ Full-text search (Russian + English)
- ✅ Metadata filtering (date, tags, source, interviewer)
- ✅ Upload and display of media attachments
- ✅ Admin panel for content management
- ✅ Clean, mobile-responsive frontend using Bootstrap 5

---

## 🛠️ Tech Stack

| Layer        | Tech              |
|--------------|-------------------|
| Backend      | Django (Python)   |
| Database     | PostgreSQL        |
| Frontend     | Django Templates + Bootstrap |
| Admin        | Django Admin      |
| Storage      | FileField (media uploads) |
| Deployment   | Firebase (in progress), Replit/GitHub |
